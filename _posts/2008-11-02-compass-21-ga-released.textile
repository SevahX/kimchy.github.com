title: Compass 2.1 GA Released
---

<a href="http://www.compass-project.org">Compass</a> version 2.1 GA released. Major feature since the RC release include the ability to configure Compass using JSON (both configuration and mappings). There are important fixes that are included, among them is the ability to work with Terracotta based index and Lucene 2.4. Full release notes can be found <a href="http://issues.compass-project.org/secure/ReleaseNote.jspa?projectId=10000&styleName=Html&version=10137">here</a>.

As always, feedback will be welcomed, and work on 2.2 M1 had already began :). Enjoy!
